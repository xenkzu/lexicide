# LEXICIDE — itch.io Upload Checklist (HTML Web Version)

## Files in this folder
| File | Purpose |
|---|---|
| `lexicide_html.zip` | Web bundle containing your `index.html` and assets. Players will play this directly in their browser on itch.io. |
| `cover.png` | Game icon/cover art to use on the itch.io page |
| `LEXICIDE.exe` | Keep this as an optional download for players who want to play offline on Windows. |

## itch.io Upload Steps

### 1. Dashboard → Create new project
- **Title**: LEXICIDE
- **Kind of project**: HTML
- **Classification**: Games
- **Genre**: Action, Typing

### 2. Uploads section
- Upload `lexicide_html.zip`.
- Click the **"This file will be played in the browser"** checkbox for the zip file.
- Check the box to auto-start or whatever preference you want.

*(Optional)*
- Also upload `LEXICIDE.exe`.
- Set platform: ✅ Windows
- Leave it as an optional Executable download.

### 3. Embed Options (Important for HTML)
- **Viewport dimensions**: `1280` x `720` (or whatever base resolution your game uses).
- Check the box for **Mobile friendly** if you eventually add mobile controls.
- Check **Fullscreen button** if you want them to easily expand it.

### 4. Details
- **Short description**: A high-speed typing action game. Type or die.
- **Description**: Write your full game description here.
- **Tags**: typing, action, webgame, word-game, gothic

### 5. Cover image
- Upload `cover.png` as the **Cover Image** (recommended: 315×250 px — itch.io will auto-crop)

### 6. Pricing
- Set to Free or Pay-What-You-Want

### 7. Visibility
- Start as **Restricted** to test the embed in your browser, make sure the audio and sprites load correctly. Then switch to **Public**.
